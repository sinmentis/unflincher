Sample data

The interactive logic runs in your browser on fictional data. After the page assets load, it requests only the committed sample file and never contacts a model. Writing, regeneration, and live chat are disabled here and available in the self-hosted app. The public site is hosted on GitHub Pages and is subject to GitHub's platform logging and privacy practices.

Explore five views

The interactive controls need JavaScript. The screenshots above show the five views.

The self-hosted app stores journal data in your local SQLite database. Generation sends the selected prompt and relevant context through GitHub Copilot. It requires GitHub Copilot access, a host you control, and basic terminal setup.
